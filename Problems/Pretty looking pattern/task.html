Pretty looking pattern

In some design styles, a 4x4 matrix pattern is considered looking pretty if it doesn't contain a 2x2 matrix of the same color. Your task is to write the program that outputs "YES" if the 4x4 matrix is looking pretty, otherwise output "NO".

The input contains 4 lines, each line contains 4 symbols, different symbols represent different colors: W stands for white color, B is for black color, R is for red, G is for green, and Y is for yellow color.


Sample Input:
WWWW
BBBB
WWWW
YYYY

Sample Output:
YES


Sample Input:
BBBB
BWWB
BWWB
BBBB

Sample Output:
NO



Memory limit: 256 MB
Time limit: 8 seconds
